Project Title :
Development of E-Commerce Website with mandatory review system

Description

Developed a complete front-end and back-end E-Commerce application. .Implemented "user authentication", "product listing", "shopping cart", and "payment gateway" integration. .Integrated a mandatory review system, ensuring customers provide feedback before completing future purchases. .Used MongoDB for database management, Node.js for backend development, and React.js for frontend UI. . Ensured a responsive and secure web application with optimized performance.

Technologies used

  • Frontend: React.js, HTML, CSS
  • Backend: Node.js, Express.js
  • Database: MongoDB

Impact

  • Improved product discovery by showing top-rated products
  • Encouraged genuine user feedback
  • Enhanced search efficiency by algorithmically promoting high-quality products
